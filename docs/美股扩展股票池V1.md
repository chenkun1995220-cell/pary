# 美股扩展股票池 V1

## 1. 目标

把最初 AAPL、MSFT、GOOGL 三只样本扩展为 15 家美股公司，使每个标准行业至少有 5 个样本，可以计算更有参考价值的行业估值中位数。

## 2. 股票池

| 行业 | 股票 |
|---|---|
| 科技软件 | MSFT、GOOGL、ORCL、CRM、ADBE |
| 科技硬件 | AAPL、NVDA、AMD、QCOM、CSCO |
| 消费零售 | AMZN、WMT、COST、HD、TGT |

种子配置：

```text
data/config/us_universe_symbols.csv
```

CIK、SEC 公司名和交易所不手工填写，由 SEC 官方 `company_tickers_exchange.json` 自动解析。

## 3. 构建与运行

设置 SEC User-Agent：

```powershell
$env:SEC_USER_AGENT='Your Name your-email@example.com'
```

构建公司配置：

```powershell
powershell.exe -ExecutionPolicy Bypass -File F:\chatgptssd\project2\scripts\build_us_universe.ps1
```

自动补齐行情：

```powershell
powershell.exe -ExecutionPolicy Bypass -File F:\chatgptssd\project2\scripts\auto_fill_us_real_sample_quotes.ps1 -Companies F:\chatgptssd\project2\data\samples\us_universe_companies.csv -Output F:\chatgptssd\project2\outputs\us_universe\market_quotes.csv
```

运行完整筛选：

```powershell
powershell.exe -ExecutionPolicy Bypass -File F:\chatgptssd\project2\scripts\run_us_real_sample.ps1 -Companies F:\chatgptssd\project2\data\samples\us_universe_companies.csv -Quotes F:\chatgptssd\project2\outputs\us_universe\market_quotes.csv -OutputRoot F:\chatgptssd\project2\outputs\us_universe
```

`market_quotes.csv` 是每周可重建的运行快照，保存在已被 Git 忽略的 `outputs/us_universe`，不作为仓库数据提交。`latest_run_summary.md` 会记录该快照的行数、行情日期范围和 SHA-256，供提交前一致性复核。

## 5. 使用边界

当前股票池是为了验证行业比较和自动化流程，不代表全部美股。ADBE、QCOM 入选只表示量化初筛通过，不构成投资建议，仍需核验财报标签、一次性项目、行业周期和估值假设。

## 6. 候选公司深研包

为候选池自动生成深研包：

```powershell
$env:SEC_USER_AGENT='Your Name your-email@example.com'
powershell.exe -ExecutionPolicy Bypass -File F:\chatgptssd\project2\scripts\generate_candidate_research_packs.ps1
```

输出目录：

```text
outputs/us_universe/research
```

每家公司报告包括估值快照、财务质量、SEC 最新 10-K/10-Q 官方链接、自动价值陷阱提示和人工核验清单。报告不对 SEC 文件内容做无依据摘要，最终判断仍需人工阅读原始申报文件。
