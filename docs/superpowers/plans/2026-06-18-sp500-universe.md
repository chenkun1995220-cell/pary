# 标普 500 全量股票池实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将现有 15 家美股股票池升级为可每周更新、可缓存回退的标普 500 全量股票池，并让约 500 家公司的 SEC 数据在一次流程中共享缓存。

**Architecture:** 新增独立成分股模块负责 MediaWiki HTML 的解析、校验和最后有效缓存；现有公司构建器继续负责与 SEC ticker/exchange 名单核对。Company Facts 缓存下沉到 `sec_edgar_adapter.load_company_facts`，三个消费模块通过同一目录复用。

**Tech Stack:** Python 3 标准库、CSV/JSON、`html.parser`、PowerShell、`unittest`
