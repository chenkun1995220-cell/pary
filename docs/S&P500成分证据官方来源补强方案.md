# S&P 500 历史证据上限与受限回测方案

## 当前结论

自 2026-07-11 起，S&P 500 历史成分证据状态正式改为：

> 证据上限已确认，仅保留受限回测。

现有审计结果仍保留：verified 比例约 15.6%，历史证据缺口 425 个，weak / secondary 行及受影响周次继续记录。关闭任务不代表证据已经补齐，也不允许把二手来源升级为 verified。

## 关闭依据

- S&P Global 公开页面无法稳定导出完整历史成分数据。
- 仅依赖指数公告倒推无法证明历史全量完整性。
- Wikipedia、Yahoo、Kaggle、GitHub 清单只能作为二手交叉验证。
- SPY、IVV、VOO 是 ETF 持仓，不能替代官方指数历史成分。
- 当前已确认的信息不足以把正式回测样本扩大到未验证历史区间。

权威策略文件：`data/config/sp500_historical_evidence_policy.json`。

## 固定边界

- `evidence_ceiling_status=evidence_ceiling_confirmed`
- `backtest_mode=limited_verified_only`
- `recurring_supplement_request_enabled=false`
- `backtest_sample_expansion_allowed=false`
- `historical_membership_auto_update_allowed=false`
- `formal_model_upgrade_allowed=false`
- `formal_model_change_allowed=false`

缺口数量保留在 `membership_evidence_unresolved_gap_count`，但不再计入 `membership_evidence_action_required_count`，也不再生成每周补证队列。

## 周度行为

每周仍运行：

- 当前成分股交叉校验；
- 严格时点回测；
- 回测证据复核；
- 统一结论、交付检查和提交前复核。

每周不再自动运行历史补证 import plan、supplement queue / batch、source intake、official export probe、verified source plan、apply preview / confirmation / approved apply plan。

这些工具保留为历史人工工具，但不会自动进入周任务，也不会生成待办。

## 当前成分股边界

历史证据任务关闭不影响当前成分名单。当前 S&P 500 成分股继续使用 Wikipedia、Yahoo、GitHub 清单与 SPY / IVV / VOO 持仓交叉校验生成替代来源，保持 `membership_evidence=secondary`，只用于当前筛选，不用于扩大历史回测样本。

## 重新开启条件

只有用户提供新的、可验证的 S&P Global 官方历史成分文件或同等级证据，并经过独立人工复核后，才能通过修改策略文件版本重新开启。周度自动化不得自行重新开启补证任务。
