# 美股/港股/A股低估公司筛选标准

> 版本：2026-06-16  
> 用途：建立跨美股、港股、A股的统一低估筛选框架，并配套 Excel 模板进行初筛。  
> 提醒：本框架用于研究和候选池筛选，不构成买卖建议。低估必须结合后续个股深度研究、风险承受能力和仓位管理。

## 1. 核心定义

本框架中的“低估公司”不是简单指低市盈率、低市净率或股价下跌，而是指：

1. 当前价格相对其盈利能力、现金流、资产质量和长期竞争力存在折价；
2. 折价不是由重大经营恶化、财务造假、治理失控、退市风险或不可逆行业衰退造成；
3. 公司具备一定安全边际，且未来 2-3 年有基本面修复、现金回报、分红回购、行业周期回升或估值均值回归的可能。

一句话标准：**价格便宜只是入口，质量和现金流决定能不能进入候选池。**

## 2. 股票池范围

### 2.1 覆盖市场

| 市场 | 主要范围 | 代码/上市地说明 |
|---|---|---|
| 美股 | NYSE、Nasdaq、AMEX 等主要美国上市普通股、ADR | 优先使用 SEC EDGAR、交易所和公司公告数据 |
| 港股 | 港交所主板、GEM、H 股、红筹、部分中概回港公司 | 注意港币计价、会计准则差异、流动性差异 |
| A股 | 上交所、深交所、北交所普通股 | 包括主板、科创板、创业板、北交所，需特别标记 ST、退市整理、未盈利标识 |

### 2.2 第一轮排除对象

初筛时建议先剔除以下公司：

1. 最近一期审计意见为非标准无保留，或存在重大持续经营疑虑；
2. 已被交易所实施退市风险警示、长期停牌、流动性极差；
3. 最近 3 年主营业务收入大幅萎缩且无法解释；
4. 经营现金流长期为负且依靠融资维持运营；
5. 资产负债率、净负债/EBITDA、短债压力明显高于行业合理区间；
6. 财务数据不可比、重大重组刚完成、收入确认方式发生重大变化；
7. 金融、地产、资源周期、早期生物科技等特殊行业未使用专门口径时，不与普通制造/消费/科技公司直接比较。

## 3. 数据整合口径

### 3.1 必备基础字段

| 类别 | 字段 |
|---|---|
| 证券信息 | 市场、股票代码、公司名称、行业、上市地、交易货币 |
| 市值估值 | 股价、市值、企业价值 EV、PE、PB、PS、EV/EBITDA、股息率 |
| 盈利能力 | 收入、毛利率、净利率、ROE、ROIC、EBITDA Margin |
| 现金流 | 经营现金流、资本开支、自由现金流、FCF 收益率 |
| 资产负债 | 资产负债率、净负债/EBITDA、流动比率、现金及等价物 |
| 成长稳定 | 3 年收入 CAGR、3 年净利润 CAGR、收入波动、利润波动 |
| 风险治理 | 审计意见、是否 ST/退市风险、商誉/净资产、股权质押、关联交易、诉讼监管 |
| 数据追踪 | 报告期、数据日期、来源链接、备注 |

### 3.2 跨市场处理原则

1. **货币统一**：估值和现金流建议统一换算为人民币或美元；表格模板允许保留原币，但比较时必须同币种。
2. **TTM 优先**：PE、PS、EV/EBITDA、FCF 收益率优先使用最近 12 个月口径；无法取得 TTM 时才用最近年度。
3. **行业内比较**：低估判断必须与同市场、同行业、同商业模式公司比较。A股消费公司不能直接与美股 SaaS 公司用同一阈值比较。
4. **金融行业单独建模**：银行、保险、券商不适合用 EV/EBITDA、FCF 收益率作为主指标，应优先使用 PB、ROE、不良率、拨备覆盖率、资本充足率等。
5. **周期行业单独看周期位置**：钢铁、有色、化工、航运、煤炭、油气等低 PE 可能是盈利周期顶部，必须结合商品价格、库存和资本开支周期。
6. **亏损成长股单独处理**：未盈利科技、生物医药、平台型公司不能只因 PE 为空就排除，但必须提高现金消耗、收入质量和融资风险权重。

## 4. 低估评分模型

总分 100 分，建议 70 分以上进入重点观察，80 分以上进入深度研究，90 分以上才考虑作为高优先级候选。

| 维度 | 权重 | 判断重点 |
|---|---:|---|
| 估值折价 | 30 | PE、PB、EV/EBITDA、FCF 收益率是否低于行业中位数，且折价有合理安全边际 |
| 盈利质量 | 25 | ROE、ROIC、毛利率、净利率是否稳定且高于行业平均 |
| 资产负债安全 | 15 | 负债率、净负债/EBITDA、流动比率是否安全 |
| 现金流质量 | 15 | 经营现金流是否覆盖净利润，自由现金流是否为正，FCF 收益率是否有吸引力 |
| 成长与稳定 | 10 | 收入和利润是否长期增长或处于可解释的周期底部 |
| 治理与流动性风险 | 5 | 审计、退市、股权质押、关联交易、流动性风险是否可控 |

### 4.1 估值折价评分

估值折价不是越低越好。极低估值需要反向检查是否存在价值陷阱。

| 条件 | 参考分 |
|---|---:|
| PE、PB、EV/EBITDA 至少两项低于行业中位数 30% 以上，且 FCF 收益率为正 | 24-30 |
| 主要估值指标低于行业中位数 15%-30%，且盈利现金流稳定 | 18-23 |
| 估值接近行业中位数，但质量明显更好 | 12-17 |
| 估值便宜但现金流差、利润波动大或负债高 | 6-11 |
| 估值便宜主要由基本面恶化导致 | 0-5 |

### 4.2 盈利质量评分

| 条件 | 参考分 |
|---|---:|
| ROIC、ROE、净利率长期高于行业，且波动较低 | 20-25 |
| 盈利能力略高于行业，近 3 年保持稳定 | 15-19 |
| 盈利能力普通但没有明显恶化 | 9-14 |
| 盈利能力下降，或利润主要来自非经常性收益 | 4-8 |
| 持续亏损且商业模式尚未证明 | 0-3 |

### 4.3 资产负债安全评分

| 条件 | 参考分 |
|---|---:|
| 净现金或低杠杆，短债压力小，流动性充足 | 12-15 |
| 杠杆处于行业正常水平 | 8-11 |
| 杠杆偏高但现金流能够覆盖 | 4-7 |
| 债务压力明显，存在再融资依赖 | 0-3 |

### 4.4 现金流质量评分

| 条件 | 参考分 |
|---|---:|
| 经营现金流持续为正，自由现金流稳定，FCF 收益率高 | 12-15 |
| 现金流总体良好，偶有资本开支波动 | 8-11 |
| 经营现金流不稳定，但有合理解释 | 4-7 |
| 长期自由现金流为负，融资依赖强 | 0-3 |

### 4.5 成长与稳定评分

| 条件 | 参考分 |
|---|---:|
| 收入和利润 3 年 CAGR 稳定为正，且行业空间未明显收缩 | 8-10 |
| 收入增长稳定但利润受周期影响 | 5-7 |
| 增长停滞但分红/回购稳定 | 3-4 |
| 收入和利润长期下滑 | 0-2 |

### 4.6 治理与风险评分

| 条件 | 参考分 |
|---|---:|
| 审计清洁、披露稳定、无重大治理风险、流动性好 | 4-5 |
| 存在轻微治理或流动性折价但可跟踪 | 2-3 |
| 审计、监管、退市、质押、诉讼等风险突出 | 0-1 |

## 5. 候选等级

| 总分 | 等级 | 动作 |
|---:|---|---|
| 90-100 | A：高优先级深研 | 进入公司年报、电话会、行业比较、估值模型 |
| 80-89 | B：重点观察 | 进入跟踪池，等待价格或基本面确认 |
| 70-79 | C：普通观察 | 保留但不优先，占用精力较少 |
| 60-69 | D：暂不深研 | 只在行业变化或估值进一步下修时复看 |
| 0-59 | E：剔除 | 不进入候选池 |

## 6. 价值陷阱检查清单

低估候选必须额外检查以下问题。任一项为“严重”，即使总分较高也应降级。

1. 行业是否处于长期不可逆衰退；
2. 公司收入是否依赖单一客户、单一产品、单一政策补贴；
3. 利润是否依赖公允价值变动、投资收益、资产处置等非经常性项目；
4. 应收账款、存货、合同资产是否大幅快于收入增长；
5. 商誉、资本化研发、长期资产减值风险是否过高；
6. 管理层是否频繁变动，是否有监管处罚或重大诉讼；
7. 分红和回购是否真实可持续，还是一次性行为；
8. 大股东质押、减持、关联交易是否明显损害中小股东；
9. 低估是否只是因为市场流动性不足，退出困难；
10. 是否存在会计准则、汇率、税务、地缘政治或制裁风险。

## 7. 不同行业的重点指标

| 行业类型 | 优先指标 | 慎用指标 |
|---|---|---|
| 银行 | PB、ROE、不良率、拨备覆盖率、净息差、资本充足率 | PE、EV/EBITDA、普通 FCF |
| 保险 | P/EV、PB、ROE、偿付能力、投资收益率 | 普通 PE |
| 券商 | PB、ROE、经纪/投行/资管结构、资本充足 | 单年 PE |
| 地产 | 净负债率、现金短债比、销售回款、土地储备质量 | 表面 PB |
| 周期资源 | PB、EV/EBITDA、成本曲线、库存周期、商品价格位置 | 周期顶部低 PE |
| 制造业 | ROIC、毛利率、FCF、产能利用率、应收存货周转 | 单一年度净利率 |
| 消费 | ROE、毛利率、品牌力、渠道库存、现金流、分红 | 短期 PE |
| 科技软件 | 收入增长、毛利率、续费率、Rule of 40、FCF | 未盈利阶段 PE |
| 医药生物 | 管线、研发效率、现金消耗、专利期、医保/监管风险 | 静态 PE |
| 公用事业 | 股息率、资产回报、监管回报、负债期限结构 | 高增长指标 |

## 9. 推荐工作流

1. 建立全市场股票池：分市场抓取或导入股票代码、公司名称、行业、市值、上市地。
2. 导入最近年度和 TTM 财务数据：收入、利润、现金流、资产负债表、股本、市值。
3. 标准化字段：统一货币、口径、报告期、行业分类。
4. 第一轮硬剔除：退市、ST、审计异常、财务不可比、极低流动性。
5. 计算相对估值：与同行业中位数比较，而不是跨行业绝对比较。
6. 计算六维评分：估值、盈利质量、资产负债、现金流、成长、治理风险。
7. 输出候选池：按 A/B/C/D/E 等级排序。
8. 对 A/B 等级公司做深度研究：年报、业务结构、行业格局、管理层、估值模型。

## 10. 使用表格模板的建议

配套 Excel 模板包含以下工作表：

1. `封面`：模型说明、评分等级、使用步骤；
2. `输入与打分`：录入股票数据并自动计算评分；
3. `行业阈值`：维护不同行业的估值参考中位数和风险阈值；
4. `数据字典`：说明每个字段的定义和录入口径；
5. `来源与口径`：记录数据来源、链接、报告期和更新日期；
6. `检查`：检查缺失值、异常分数和高风险标记。

第一版建议先人工导入 50-200 只候选股票测试模型稳定性。等筛选逻辑稳定后，再扩展为自动化抓取和全市场覆盖。
